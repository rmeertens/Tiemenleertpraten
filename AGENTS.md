# Projectdoel
Bouw een minimalistische leerwebtool voor logopediestudenten waarmee zij toetsgericht kunnen trainen richting een 8-10.

# Ontwerpprincipes
- Minder opties, meer flow.
- Alles koppelen aan beoordelingscriteria.
- Geen lange samenvattingen zonder oefenvraag.
- Elke uitleg eindigt in toepassing.
- Feedback is streng, concreet en toetsgericht.
- Bouw als trainingsmachine, niet als bibliotheek.

# Kernflow
Bron -> leerdoel -> oefening -> feedback -> herhaling -> toetsklaar-score.

# Veiligheid
- Geen auteursrechtelijke boeken, artikelen of casusdocumenten openbaar tonen.
- Gebruik alleen korte, eigen samenvattingen en geparafraseerde trainingsdata.
- Reproduceer geen beschermde toetsvragen letterlijk.
- Zet persoonlijke of herleidbare gegevens niet in publieke code.
