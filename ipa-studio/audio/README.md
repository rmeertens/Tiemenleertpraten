# Inspreeklijst voor de IPA Studio

Hier komen jouw audio-opnames. Zolang een bestand er nog niet staat, toont de app netjes "audio nog niet ingesproken" — niets breekt.

## Bestandsformaat

- **Format:** MP3 (44.1 kHz, mono, ±128 kbps is ruim genoeg).
- **Lengte:** isoleer de klank tussen ±0,5 sec stilte.
- **Naamgeving:** exact zoals hieronder. Hoofdletter-gevoelig.
- **Locatie:** zet ze direct in deze map (`ipa-studio/audio/`).

## Tips voor het inspreken

- Stille kamer, microfoon ±15 cm afstand, recht voor je mond.
- Spreek elke klank los, **niet** in een woord. Voor klinkers: aanhouden ±0,7 sec. Voor explosieven (/p/, /t/, /k/, /b/, /d/, /g/): één duidelijke plof.
- Voor stemloze fricatieven (/s/, /f/, /x/) ±0,5 sec aanhouden.
- Neem alles in één sessie op met dezelfde mic en afstand — anders klinkt het rommelig.
- Tools die werken: **Audacity** (gratis), **GarageBand** (Mac), of zelfs de iPhone-app **Spraakmemo's** in een stille kamer.
